# Bitcoin Price Analysis Python

This is a simple Python project for analyzing Bitcoin price data from a CSV file.

The script reads `BTC-USD.csv`, shows basic information about the dataset, calculates simple statistics for the `Open` and `Close` prices, and plots the Bitcoin closing price over time.

## Files

- `btc_exercise.py` - Python script for analysis and visualization
- `BTC-USD.csv` - Bitcoin historical price dataset

## Requirements

```bash
pip install pandas matplotlib
```

## Run

```bash
python btc_exercise.py
```
